# H1

## Pitch - Hvad har i lavet og hvilke teknologier har i brugt (blazor + den database i bruger) - Maks 150 ord

Pure Aura er en webapplikation hvor der er man kan oprette en gruppe, upload produkter, giv feedback på et produkt (kun 5 stjerner), tilføje produkter til ens indkøbskurv eller gemme dem til ens wishlist. Bruger og produktdannelse fungerer ved at anvende en Neon.Postgres database, hvor vi sender vores kommandoer og data. 

Anvendte teknologi: 

•	Blazor (.NET) 
•	C# 
•	.Razor komponenter 
•	PostgreSQL 
•	NpgSQL (Nuget Package) 
•	HTML /CSS / Bootstrap 
•	Authentication (Log in / Sign up)

## Opstartsguide af projektet - Hvordan starter man det!
For at kunne starte vores applikation skal man havde gjordt følgende først:
•	Installere .NET SDK
•	Installere Npgsql Nuget Package 
•	Installere Visual Studio 2026 (Der bruges funktioner fra .NET 10) 
Efter det er bleven installeret, skal følgende gøres for at starte den: 
•	Åbn Visual Studio 2026 
•	Åbn Pura.slnx løsningen 
•	Køre vi https som åbner en lokalhost på port 7239.
